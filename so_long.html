Game Development & 2D Graphics

So Long

Jeu 2D de collecte et d'évasion avec rendu graphique en C

À propos du projet

So Long est un jeu 2D de type top-down développé en C avec la bibliothèque MinilibX dans le cadre de l'école 42. Le joueur doit collecter tous les objets sur la carte avant de pouvoir s'échapper par la sortie, tout en évitant les ennemis. Ce projet explore les bases du développement de jeux vidéo et du rendu graphique 2D.

Fonctionnalités principales

  • Rendu graphique 2D

    Affichage de sprites, de tuiles et d'animations avec la bibliothèque MinilibX

  • Système de collecte

    Mécanique de jeu basée sur la collecte d'objets pour débloquer la sortie

  • Parser de carte

    Lecture et validation de fichiers .ber pour charger les niveaux

  • Gestion des collisions

    Détection des collisions avec les murs, objets et ennemis

  • Compteur de mouvements

    Affichage en temps réel du nombre de déplacements effectués

Défis techniques

Le principal défi était de gérer efficacement les événements clavier et le rendu graphique sans utiliser de moteur de jeu. La création d'un parser de carte robuste capable de valider différents formats et de gérer les erreurs a également été un aspect important. L'optimisation du rendu pour maintenir des performances fluides avec de nombreux sprites à l'écran a nécessité une attention particulière.
